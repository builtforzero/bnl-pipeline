N/A

Requirement


Required columns in your file must have values in the right format. Specifically:

  • DATES must be in the format "MM/DD/YYYY" e.g. "12/31/2020".
  • NUMERICAL FIELDS like Age and Household Size must only contain whole numbers e.g. "31".
  • OPEN TEXT FIELDS like Race and Gender can accept any combination of letters, numbers, and special characters.
